# StudyFlow

Simple React application deployed with GitHub Pages.

## Public URL

`https://erimar2678ecpi.github.io/5.6PA-school-assignment`

## Repository

`https://github.com/erimar2678ecpi/5.6PA-school-assignment`

## Deployment Setup

- Platform: GitHub Pages
- React base path: `homepage` in `package.json`
- Deployment package: `gh-pages`
- Deploy command: `npm run deploy`

## Scripts

- `npm start` runs the local development server.
- `npm test -- --watch=false` runs the test suite once.
- `npm run build` creates the production build.
- `npm run deploy` builds the app and publishes the `build/` folder to the `gh-pages` branch.

## Notes

- The app is a single-page responsive personal dashboard called `StudyFlow`.
- GitHub Pages hosting is configured for the repository path `/5.6PA-school-assignment/`.
